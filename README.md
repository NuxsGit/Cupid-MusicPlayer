# Cupid 🎧

Cupid is a high-performance mobile music player built with Tauri 2.0 and Svelte. It features a lightweight, dynamic system theme manager that adapts flawlessly while maintaining 60 FPS battery efficiency. Powered by a Rust backend, Cupid handles audio decoding natively in the background for absolute stability.

## 🛠️ Tech Stack

* **Frontend:** Svelte + TypeScript (optimized for ultra-low overhead and smooth theme switching)
* **Backend:** Rust + Tauri 2.0 (native audio management, file system access, and system-level performance)
* **Audio Engine:** Native hardware acceleration via Rust
